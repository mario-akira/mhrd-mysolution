# mhrd-mysolution

Solutions to MHRD, a game by Funghisoft

This is a possible solution to the entire game. Don't expect major optimizations; it was done with several breaks in reasoning...my dogs need walks.

Special thanks to @edwardbarak for creating a code extractor for the game save.
